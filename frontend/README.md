# B-ware Landing Page Frontend

This is the frontend code bundle for the B-ware landing page. It is built as a Single Page Application using modern web technologies.

## Tech Stack

This project is built using:
- **React (v18)** - Frontend framework
- **Vite** - Build tool and development server
- **Tailwind CSS** - Utility-first CSS framework
- **Material UI (MUI)** & **Radix UI** - Component libraries for accessible, pre-built UI elements
- **Lucide React** - Icon library
- **React Router** - Navigation and routing
- **Framer Motion & tw-animate-css** - Animations

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You need Node.js installed on your machine. You can use either `npm` or `pnpm` (pnpm is recommended in the `package.json` overrides).

### Installation & Running Locally

1. **Install Dependencies:**
   Run the following command in the root directory to install all necessary packages:
   ```bash
   npm install
   # or
   pnpm install
   ```

2. **Start the Development Server:**
   Once the dependencies are installed, start the local server:
   ```bash
   npm run dev
   # or
   pnpm dev
   ```

3. **View the App:**
   Open your browser and navigate to `http://localhost:5173/` (or whichever port Vite outputs in the terminal).

## Building for Production

To create a production-ready build, run:
```bash
npm run build
# or
pnpm build
```
This will compile the application into the `dist/` directory, which can then be deployed to any static hosting service.
